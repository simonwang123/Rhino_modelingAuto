# Model Exchange Workspace

Use this directory for Rhino-to-APDL exchange files.

```text
model_exchange/
  rhino_models/       Rhino source .3dm models for APDL export
  apdl_exports/       Generated SAT files, manifests, reports, and APDL macros
```

Generated SAT and APDL job files can be recreated from Rhino source models.
